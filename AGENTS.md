# AGENTS.md

This file provides instructions and available skills for AI coding agents working in this repository.

## Project Purpose

khmerskills is a universal skills loader for AI coding agents. It installs skills from GitHub repositories and synchronizes discoverable skill metadata into AGENTS.md.

## Agent Operating Rules

- Prefer small, focused changes.
- Preserve existing CLI behavior unless a change is explicitly requested.
- Keep TypeScript strict-mode compatible.
- Run `npm run build` after code changes.
- Do not remove user-authored content outside requested scope.

## Core Commands

- Install skill repo: `khmerskills install owner/repo`
- List installed skills: `khmerskills list`
- Read one skill: `khmerskills read <skill-name>`
- Sync AGENTS file: `khmerskills sync`
- Remove a skill: `khmerskills remove <skill-name>`

## How to Use Skills

To load a specific skill, run:

```bash
khmerskills read <skill-name>
```

Example:

```bash
khmerskills read code-review
```

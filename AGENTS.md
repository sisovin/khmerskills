# AGENTS.md

This file provides instructions and available skills for AI coding agents working in this repository.

## Project Purpose

khmetskills is a universal skills loader for AI coding agents. It installs skills from GitHub repositories and synchronizes discoverable skill metadata into AGENTS.md.

## Agent Operating Rules

- Prefer small, focused changes.
- Preserve existing CLI behavior unless a change is explicitly requested.
- Keep TypeScript strict-mode compatible.
- Run `npm run build` after code changes.
- Do not remove user-authored content outside requested scope.

## Core Commands

- Install skill repo: `khmetskills install owner/repo`
- List installed skills: `khmetskills list`
- Read one skill: `khmetskills read <skill-name>`
- Sync AGENTS file: `khmetskills sync`
- Remove a skill: `khmetskills remove <skill-name>`
